# 05. АРХИТЕКТУРА ИНТЕРФЕЙСА (15 ЭКРАНОВ)

## 5.1. Блок А: Поиск и Потребление
* **L1 (Информация)**: Экраны для навигации клиента.
* **L2 (Пояснение)**: 
  - **Discovery (Главная)**: Ленты-карусели, Стэкинг карточек бизнеса, 7 секторов.
  - **Search (Результаты)**: Семантический поиск с фильтрами цены/расстояния внутри кругов Данбара.
  - **Map (Карта)**: Золотые пины (Данбар) и Серебряные пины (Глобал) с цифрами рейтинга.
  - **Favorites (Избранное)**: Сохраненные карточки мастеров, сгруппированные по секторам.

## 5.2. Блок Б: Профили и Коммуникация
* **L1 (Информация)**: Экраны взаимодействия с мастером.
* **L2 (Пояснение)**: 
  - **Public Profile**: Витрина мастера с ABCD-аналитикой, QR-визиткой и списком услуг.
  - **Service Detail**: Детальная карточка услуги (Фото, 200 симв. описания, цена).
  - **Lead Form**: Динамическая форма заявки. Генерирует поля (Date, Time, Qty, Period) на базе флагов карточки. Сохраняет `Lead` в БД и предлагает переход в Telegram Master Chat с предзаполненным контекстом.
  - **QR Scanner**: Встроенный сканер для сделок и знакомств.

## 5.3. Блок В: Личный кабинет и Финансы
* **L1 (Информация)**: Управление данными пользователя и кошельком.
* **L2 (Пояснение)**: 
  - **My Profile**: Редактирование био, чекбоксы юридических оферт.
  - **Wallet (Кошелек)**: Единый баланс ⭐️, курс Оракула, кнопки Ввод/Вывод TON.
  - **Referral Tree**: Дерево L1-L3, B2B доход, история кэшбэка.
  - **Dunbar Radar**: 3 интерактивных кольца и Shadow List (Drag-and-drop).

## 5.4. Блок Г: Мастерская (Бизнес-режим)
* **L1 (Информация)**: Инструменты для ведения бизнеса.
* **L2 (Пояснение)**: 
  - **Business Landing**: Мини-лендинг перед созданием бизнеса, стильно оформленный, описывающий преимущества платформы для продавцов.
  - **Master Dashboard**: Графики просмотров услуг (счетчики "глазик"), конверсии, лиды.
  - **Card Editor (MasterCardBuilder)**: Конструктор до 20 услуг (Сектор, цена, фото, описание) с автоматической логикой флагов контекста (Service, Item, Event, Dining, Rental, SOS).
  - **Quick Checkout**: POS-терминал. Поля Сумма + Мемо -> генерация QR.

## 5.5. Блок Д: Системные и Глобальные элементы
* **L1 (Информация)**: Элементы интерфейса, доступные сквозь приложение.
* **L2 (Пояснение)**: 
  - **Global Tooltips (Info)**: Всплывающие подсказки (попапы) со стилем глассморфизма, скрытые под иконками (i) по всему интерфейсу, поясняющие сложную механику.
  - **Add User Screen**: Экран поиска и добавления новых пользователей в свой Граф Доверия (активируется по кнопке "+" из центра Радара).

## 5.6. Обновления UI (Спринт 8)
- **Master Profile**: Блок оценки ABCD скрыт под попап (4 горизонтальные шкалы), вызываемый при нажатии на рейтинг.
- **Radar Screen**: Кольца растянуты на всю ширину экрана. В центре добавлен круг с плюсом "+" для перехода на экран `Add User Screen`.
- **Discovery Screen**: Карточки переработаны для акцента на фото, названии и рейтинге. Нажатие ведет напрямую в карточку Мастера.

## 5.7. Стрикт-дизайн (Telegram Native)
Для обеспечения премиального пользовательского опыта и нативной интеграции с Telegram Mini Apps, соблюдаются следующие стандарты:
- **Типографика**: Уменьшенные размеры шрифтов. Заголовки `text-xl` (вместо 2xl), основной текст `text-sm` (вместо base).
- **Сетки и отступы**: Плотная компоновка. Паддинги `p-3` и `p-4` вместо стандартных 6/8. Гапы `gap-3/4`.
- **Элементы управления**: Слайдеры и переключатели реализованы в стиле iOS/Telegram. Высота стандартных кнопок `h-10` или `h-12`.
- **Закругления**: Системное закругление углов `rounded-xl` (12px-16px) для карточек и кнопок.
- **Глассморфизм**: Использование `backdrop-blur-xl` и полупрозрачных фонов `bg-tg-secondary/70` для всех оверлеев и модальных окон.
